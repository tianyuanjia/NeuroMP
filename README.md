# NeuroMP

## Usage
**Step 1: Generate the training data**  
*Example:*  
```bash
python data/generate_env_prm.py
```

**Step 2: Train the Selector**

```bash
python train_selector.py
```

**Step 3: Train the Heuristic**

```bash
python train_heuristic.py
```

**Step 4: Evaluation**
```bash
python main.py
```
